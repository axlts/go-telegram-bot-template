# go-telegram-bot-template

[![go.mod](https://img.shields.io/github/go-mod/go-version/axlts/go-telegram-bot-template)](go.mod)
[![Go Report Card](https://goreportcard.com/badge/github.com/axlts/go-telegram-bot-template)](https://goreportcard.com/report/github.com/axlts/go-telegram-bot-template)

A simple and lightweight template for building Telegram bots in Go. This repository provides a straightforward starting
point, allowing you to quickly set up and customize your own bot with minimal overhead. Ideal for developers looking for
an easy-to-use framework to kickstart their Telegram bot projects without unnecessary complexity.

---

**Note:** This project is still a work in progress (WIP), and more features and improvements are coming soon.
